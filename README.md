UI--Action-Bar---Action-Overflow-button-is-missing-from-the-Action-Bar-on-the-Galaxy-SII-and-SIII.
==================================================================================================

Expected Result: The Action Bar will be visible on every page and contain the page title, search button, menu button and action overflow button.  Actual Result: On the Galaxy SII and SIII phones the Action Bar on every page of the Aereo App is missing the Action Overflow button.  Steps to Reproduce: 	1. Load the Aereo App (AereoMay29th.apk) on a Galaxy SII or SIII. 	2. Go to any page within the App. 	3. Inspect the Action Bar. 	4. Observe that the Action Overflow button is missing from the Action Bar.
